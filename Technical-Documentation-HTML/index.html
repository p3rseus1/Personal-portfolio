Introducao

Nessa documentação, iremos passar para você como funciona a criação de um site. Como é a sua estrutura e algumas tags básicas

Esperamos que com isso, você possa desenvolver cada vez mais suas habilidades para o desenvolvimento WEB. Boa leitura!

O HTML é uma linguagem de marcação usada pelos os navegadores para fazer a estrutura de um site. Junto com o CSS e JavaScript, ele formam um grupo poderoso para a criação de um Web Site.

Como funciona a internet

A internet foi uma das maiores invenções da humanidade. Mas você sabe como funciona?

  • Basicamente, o cliente, no caso você e seu computador, requisita uma página no seu navegador escrevendo o site na barra de navegação.
  • Essa requisição primeiro vai para um servidor DNS, que é reponsável por achar o real endereço desse site (um endereço IP).
  • Quando é encontrado esse real endereço, a sua requisição é enviada para um servidor WEB, que manda o site que você quer acessar.

Estrutura basica de um documento HTML

A estrutura de um site HTML é feito de maneira hieráquica, com uma tag dentro de outra.

A estrutura básica seria feita da seguinte maneira:

  • DOCTYPE html: essa tag determina para um browser qual é o tipo de documento que ela é.
  • HTML: essa tag é onde fica o que esse browser irá ler.
  • Head: fica algumas informações básicas do documento. Ex.: link para a estilização, a linguagem usada, tags para otimização de SEO, e outros.
  • Body: é onde fica a estrutura do seu site, como textos, imagens, links e etc.

Um exemplo prático de uma estrutura HTML:

            <!DOCTYPE html>
              <html>
                <head>
                    <title>Título da página</title>
                    <meta charset="utf-8"/>
                </head>
                <body>

                </body>
              </html>

Seu primeiro Hello world

O primeiro "Hello, world!" ou "Olá, mundo!" na vida de um programador, é inesquecível. É lá onde nos temos um entusiasmo para seguirmos.

No código a seguir, iremos te ensinar como fazer o seu primeiro "Hello, world!". Não se preocupe com a tag dentro do body, iremos te ensinar no decorrer do documento. 

              <!DOCTYPE html>
                <html>
                  <head>
                      <title>Título da página</title>
                      <meta charset="utf-8"/>
                  </head>
                  <body>
                    <h1>
                      Hello, world!
                    </h1>
                  </body>
                </html>
